{% load static %} Sistema de Eventos e Agendamentos

Estamos atualmente no processo de implementação de um sistema simples de eventos e agendamentos.

O sistema foi projetado para facilitar a gestão e o acompanhamento de eventos, permitindo que os usuários se cadastrem, façam login e realizem várias ações relacionadas aos eventos.

Funcionalidades Atuais

  • Login e Cadastro de Usuários: O sistema oferece um processo simples de login para usuários registrados, além de permitir novos cadastros, garantindo que apenas usuários autenticados tenham acesso às funcionalidades principais.
  • Cadastro de Eventos: Usuários podem criar novos eventos, definindo detalhes importantes como data, hora, descrição e local.
  • Edição e Exclusão de Eventos: Os eventos cadastrados podem ser facilmente editados ou excluídos, oferecendo flexibilidade para ajustar informações ou remover eventos desnecessários.
  • Visualização de Eventos: Os usuários têm acesso a uma visualização clara e acessível dos eventos cadastrados, com a possibilidade de visualizar os detalhes de cada evento.

Este sistema está em fase de implementação, e em breve novas funcionalidades serão adicionadas, como a integração de notificações, sistemas de lembretes e visualizações de calendário.
